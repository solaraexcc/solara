## 🛠️ Troubleshooting Common Issues

**❌ Solara Executor won't open on Windows**
The most common cause is Windows Defender quarantining the executable during extraction.
**Solution:** Add the **Solara Executor** folder (e.g. `C:/Desktop/SolaraExecutor`) to your antivirus exclusion list, then re-download the installer.

**❌ Injection fails with "Timeout" error**
This usually means Roblox updated and the current offset table is stale.
**Solution:** Check the **Solara Executor Status** page for a patch ETA, then update to the latest build once available.

**❌ Solara Executor Apk crashes on launch**
Older Android WebView versions can cause rendering failures in the UI layer.
**Solution:** Update Android System WebView from the Play Store, then relaunch the **Apk Solara Executor** app.

**❌ Solara Executor Ios Ipa won't install via AltStore**
Expired signing certificates or mismatched Apple IDs cause sideload failures.
**Solution:** Refresh your AltStore certificate, ensure your Apple ID matches, and re-sign the **Solara Executor Ios Ipa** file.

**❌ Scripts execute but nothing happens in-game**
The script may target a different game version or use deprecated API calls.
**Solution:** Verify script compatibility in the Script Hub, or enable `debug` logging to inspect execution output.
